GitHub

There can be diffrent versions of git but one of the most common is Github

In the Terminal there are basic useful commands for using git:

  1. git status => get status of current project
  2. git add . => add all
  3. git commit -m"yourComment" => set what you want to commit with a comment about what is done
  4. git push => push your change up to git
  • git pull => update your project from git

Learn more at learngitbranching.js.org
